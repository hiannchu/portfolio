Tools used: Adobe Illustrator, Photoshop, Figma

Traditional weather apps often rely on automated data that may not accurately reflect hyperlocal, real-time weather conditions, especially in outdoor settings like campgrounds, hiking trails, and remote areas. This lack of precision can make it challenging for outdoor enthusiasts—like hikers and campers—to plan their activities safely. Compared to other Australian weather apps like Weather Zone, many existing apps offer limited community-driven insights, meaning users miss out on firsthand weather reports from people who are actually on-site.

The solution aims to provide outdoor enthusiasts with a weather app to share real-time updates on weather and conditions at campsites, forests, trails, and other outdoor locations, helping them stay informed and prepared for the rapidly changing weather across Queensland.

Core Features

Real-Time Weather Updates

The homepage will present the current weather of the user's location, using large, identifiable weather icons and a light, high-contrast color palette for a stylish, easy-to-read design.

Map-View Weather Reports

Allows users to explore nearby weather conditions in real-time, helping them stay informed. The interactive map provides an overview of current conditions across various locations, ensuring users are always up to date.

Share & Update

Users can enhance the accuracy of weather updates by contributing to the community. They can create new posts by selecting a weather icon and adding helpful suggestions, as accurate weather information is crucial for outdoor enthusiasts.

Community-Drive Weather Insights

The community feature allows users to view the latest updates from recent visitors at outdoor locations like forests, national parks, campgrounds, and hiking trails. These updates ensure users stay informed.

User Flow

The user flow outlines the structure of Pixel Weather and how both guests and registered users interact with the platform. Rectangles represent different screens, while diamonds indicate interaction points—color-coded to distinguish between actions available to all users and those exclusive to registered users. Below is an overview of the four main screens: Home, Map, Add Post, and Community.

Home

The Home Screen provides real-time weather updates for the user’s current location. Users can also search for weather information in other suburbs or cities.

Map

The Map allows users to check nearby weather conditions. By clicking on an icon, users can switch to a detailed weather information view—using the same template as the Home Screen.

Add Post

This feature is designed for outdoor enthusiasts who rely on firsthand, real-time weather insights. Users can add a post by selecting a nearby location (e.g., national parks, forests, or trails) and updating the current weather conditions. Since API-provided data may not always reflect real-time conditions, this feature allows users to share accurate updates. Additionally, users can add notes to inform others about potential hazards, such as damaged trails or unsuitable camping spots.

Community

The Community section enables users to explore weather updates and posts shared by others. Users can either browse bookmarked locations or search for new ones. Each community page displays the location’s weather conditions, which can be expanded for a detailed view (using the same template as the Home Screen). Additionally, the location address is copyable for easy navigation via third-party apps.

Wireframes

The wireframes illustrate the core screens of Pixel Weather, emphasizing usability and accessibility. Below is a breakdown of each screen’s key elements:

Home Screen

The Home Screen is the first page users see, featuring a prominent weather icon to quickly convey current conditions. It also provides detailed weather data for the user’s location, including hourly forecasts and a seven-day outlook. A variety of weather icons ensure easy recognition of changing conditions.

Map Screen

Weather icons are displayed across the map, positioned based on different cities and suburbs to provide an at-a-glance view of nearby weather conditions.

Add Post Screen

When users create a new post, the location’s weather icon appears alongside the place’s name and address. This helps users compare API-retrieved weather data with actual conditions. If the information is inaccurate, users can correct it and contribute more reliable, firsthand insights for the community.

Community Screen

The Community section allows users to explore posts shared by others. The top of the page features a map displaying the general location, with the current weather conditions shown below. The latest posts are listed beneath the weather details, each accompanied by a weather icon selected by the original poster.
