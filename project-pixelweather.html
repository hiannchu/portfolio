Pixel Weather is a mobile app designed to improve weather forecasting in Queensland’s unpredictable microclimates. Traditional forecasts lack precision for short-distance predictions, which affects students and professionals relying on accurate weather updates. The app enhances forecasting by integrating conventional weather data with real-time, community-generated updates, allowing users to share and receive hyper-local weather information. By providing a more interactive and location-specific weather experience, Pixel Weather helps users make informed daily decisions, such as planning commutes or preparing for changing conditions.

Core Features

  • 1. Pixel Weather combines OpenWeather API data with community-generated updates for accurate, real-time local forecasts.
  • 2. Pixel Weather uses MySQL to store user data, enabling personalized weather alerts and allowing users to save locations for tracking.
